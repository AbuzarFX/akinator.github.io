Commands of Akinator
We provide you the best features in Akinator
Basic Commands


aki aki [person, animal, object] - Starts the Akinator Game!

aki lang - Learn how to change the language of the bot.

aki prefix - Change the prefix of Akinator!

aki ping - Returns the ping of the bot.

aki info - Returns the information of the bot.

end - Ends the current akinator game.

aki invite [inv] - Returns the invite link of Akinator.


You'll understand the rest of the game as you go with it. Have fun!
